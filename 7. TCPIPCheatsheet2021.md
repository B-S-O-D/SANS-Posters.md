[-r pcapfile] [-s snaplen] [-w pcapfile] [‘bpf filter’]
-A
display payload
-c n
display first n packets
-D
list interfaces
-e
display data link header
-F
read filter expression from file
-i
listen on specified interface
-n
do not resolve IP addresses / ports
-r
read packets from file
-s
set snap length in bytes
-S
display absolute TCP sequence numbers
-t
do not print timestamp
-tttt
print date and time
-v
verbose (multiple v: more verbose)
-w
write packets to file
-x
display in hex
-xx
display link layer in hex
-X
display in hex + ASCII
